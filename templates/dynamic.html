Dynamic Plot

Description


Prediction using a linear dynamical system

A linear dynamical system f, which has a single attractor at the correct solution and approximates the motion of students through the space of programs(orange). The X-axis represents the progress of program control while the Y-axis represents variance. Arrows are scaled with factor 0.3 for better visibility. An example trace starting at the empty program and following the dynamical system is highlighted(blue). Whenever the decoded program changes along the trace, we display the corresponding code. The coordinate system is the same as in Figure.
